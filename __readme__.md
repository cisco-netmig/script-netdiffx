# Netdiffx

## Overview
**Netdiffx** is a tool for comparing network device configurations and `show` command outputs using **Ratcliff-Obershelp similarity**. It helps identify differences based on indentation and structure.

## Features
-Compare network configurations before and after migration.
-Uses Ratcliff-Obershelp similarity for accurate text comparison.
-Generates reports for further review.
-Post-migration validation to ensure consistency.

## Usage
- Paste or browse the files for comparison.
- Click the **Compare** button to analyze differences.
- Save reports for documentation.
